### Итоговая контрольная работа

## Информация о проекте
# Необходимо организовать систему учета для питомника в котором живут домашние и вьючные животные.

# Как сдавать проект
- Для сдачи проекта необходимо создать отдельный общедоступный репозиторий.
- Разработку вести в этом репозитории, использовать пул реквесты на изменения.
- Программа должна запускаться и работать, ошибок при выполнении программы быть не должно.
- Программа, может использоваться в различных системах, поэтому необходимо разработать класс в виде конструктора.

## Задание
# Linux
1. Используя команду cat в терминале операционной системы Linux создать файлы.
1.1. Создать файл Домашние животные (заполнив файл собаками, кошками, хомяками).
1.2. Создать файл Вьючные животными заполнив файл Лошадьми, верблюдами и ослы).
1.3. Объединить созданные файлы.
1.4. Просмотреть содержимое созданного файла.
1.5. Переименовать файл в "Друзья человека".
2. Создать директорию, переместить файл туда.
3. Подключить дополнительный репозиторий MySQL.
3.1. Установить любой пакет MySQL из подключенного репозитория.
4. Установить и удалить deb-пакет с помощью dpkg.
5. Выложить историю команд в терминале ubuntu.

# Диаграммы
6. Нарисовать диаграмму, в которой есть родительский класс.
6.1. Подклассы домашние животные и вьючные животные.
6.2. Подклассы для домашних животных: собаки, кошки, хомяки.
6.3. Подклассы для вьючных животных: лошади, верблюды и ослы.

# SQL(MySQL)
7. В MySQL создать базу данных "Друзья человека". // "human_friends"
8. Создать таблицы с иерархией из диаграммы в БД.
9. Заполнить низкоуровневые таблицы:
- имена животных.
- команды которые животные выполняют.
- даты рождения животных.
10. Удалите таблицу верблюды.
10.1. Объедините таблицы "лошади" и "ослы".
11. Создать таблицу "молодые животные" (young_animals).
- Поместите в нее всех животных старше 1 года и младше 3 лет.
- В отдельном столбце с точностью до месяца подсчитайте возраст животных.
12. Объединить все таблицы в одну, при этом сохраняя поля, указывающие на прошлую принадлежность к старым таблицам.

# Программирование
13. Создать класс с Инкапсуляцией методов и наследованием по диаграмме.
14. Написать программу, имитирующую работу реестра домашних животных, со следующим функционалом:
14.1. Завести новое животное.
14.2. Определять животное в правильный класс.
14.3. Увидеть список команд, которое выполняет животное.
14.4. Обучить животное новым командам.
14.5. Реализовать навигацию по меню.
15. Создайте класс-счетчик, со следующим функционалом:
15.1. Метод add(), увеличивающий значение внутренней int переменной на 1 при нажатии "Завести новое животное".
15.2. Объект типа счетчик должен уметь работать в блоке try-with-resources.
15.3. Должно срабатывать исключение, если работа с объектом типа счетчик была не в ресурсном try и/или ресурс остался открыт.
15.4. Значение считать в ресурсе try, если при заведения животного заполнены все поля.
